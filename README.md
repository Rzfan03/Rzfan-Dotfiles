## Information
<li>
  OS : Fedora 41
  WM : i3wm 
  Terminal : Alacritty
  Compositor : Picom
  Bar : Polybar
  Shell : Zsh
  Application Launcher : Rofi
  Theme : Catpuccin Mocha
</li>


## Usage
clone the repository and copy the files to the .config directory, then you can configure it yourself.
